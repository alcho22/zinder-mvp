# Zinder MVP

Zinder is a marketplace that connects car owners with trusted local automotive service providers.

## Initial stack

- Next.js 16
- TypeScript
- Tailwind CSS 4
- Prisma
- PostgreSQL

## MVP scope

- Marketing landing page
- Service category discovery
- Request-a-service flow
- Customer dashboard
- Provider dashboard
- Quote flow
- Reviews and trust signals

## Getting started

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Database

1. Copy `.env.example` to `.env`
2. Set `DATABASE_URL`
3. Run Prisma migrations after the schema is finalized
